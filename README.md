# My dotfiles

Repositório para armazenar meus dotfiles e scripts de configuração.

## Setup

- Editor: Neovim ([lazyvim](https://www.lazyvim.org/))
- Terminal: Alacritty
  - Tema: [tokyonight](https://github.com/folke/tokyonight.nvim)
  - Emulador: [tmux](https://github.com/tmux/tmux)
- Gerenciador de janelas: Hyprland
- Shell: Zsh com Oh My Zsh
- Fontes: [Nerd Fonts](https://www.nerdfonts.com/)
- Gerenciador de pacotes: [paru](https://github.com/MakotoYui/paru)
- Sistema Operacional: Arch Linux
- Gerenciador de arquivos: [Thunar](https://docs.xfce.org/xfce/thunar/start)
